# DPAS 일일 히스토리

이 폴더는 DPAS 공부와 작업을 하루 단위로 이어가기 위한 현황판을 저장한다.

운영 규칙:
- 파일은 하루에 하나만 만든다: `YYYY-MM-DD.md`.
- 모든 내용을 하나의 거대한 파일에 계속 누적하지 않는다.
- 날짜가 바뀌면 새 파일을 만들고, 전날 파일에서는 미해결 질문과 다음 작업만 짧게 가져온다.
- 테스트 명령어, 빌드 로그, 긴 터미널 출력은 이 폴더의 핵심 내용이 아니다.
- 이 폴더는 "다음에 어디서부터 이어서 공부하고 작업할지"를 빠르게 복구하기 위한 기록이다.
- 큰 결정이나 마일스톤은 Notion에도 같이 남긴다.

권장 섹션:
- 현재 위치
- 오늘 이해한 핵심 개념
- 오늘 한 작업의 의미
- 질문과 답
- 중요한 코드 위치
- 헷갈리기 쉬운 점
- 남은 리스크
- 다음에 볼 것
